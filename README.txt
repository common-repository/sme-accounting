== Installation ==

= Modern Way: =
1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "SME Accounting".
3. Install, then Activate it.
4. Follow the [Documentation](https://smevai.com/docs)

= Old Way: =
1. Upload `sme-accounting` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the [Documentation](https://smevai.com/docs)
